# 🏺 Inferius: La Cueva del Horrocrux
<img width="1264" height="842" alt="image" src="https://github.com/user-attachments/assets/79845823-e02b-483f-af89-b2a104e17f3f" />


**Inferius** es un proyecto de aventura y exploración desarrollado en **Unity 6**. Esta experiencia sumerge al jugador en la atmósfera oscura y misteriosa de la cueva del horrocrux, enfocándose en mecánicas de movimiento fluidas, físicas realistas y una ambientación cinematográfica de alto nivel.

## 🚀 Estado Actual del Proyecto
En esta fase inicial, se ha consolidado la base técnica del personaje principal:
* **Locomoción Avanzada:** Implementación de *Blend Trees* para transiciones suaves entre Idle, Caminar y Correr.
* **Sistema de Salto Físico:** Mecánica de salto optimizada con detección de suelo (*Grounded*) mediante Character Controller.
* **Cámara Cinematográfica:** Sistema de seguimiento "Over-the-shoulder" con suavizado de movimiento.
* **Gravedad Personalizada:** Ajustes de física para simular un movimiento pesado y realista dentro de la cueva.

## 🛠️ Tecnologías y Herramientas
* **Motor:** Unity 6 (Versión 6000.0.x)
* **Lenguaje:** C# (.NET)
* **Control de Versiones:** Git + GitHub (Git LFS para Assets pesados)
* **Modelado y Animación:** Mixamo / Autodesk Maya (Rigging Humanoid)

## 🎮 Controles de Usuario
* **W, A, S, D:** Movimiento del personaje.
* **Shift Izquierdo:** Sprint (Correr).
* **Espacio:** Saltar (Soporta salto en carrera).
* **Mouse:** Control de la perspectiva de cámara.
